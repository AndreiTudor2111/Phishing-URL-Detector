# Research Papers

This directory contains 8 academic research papers that form the theoretical foundation for this phishing detection system.

## Papers Included

### 1. Classifying Phishing URLs Using Recurrent Neural Networks
- **Focus**: LSTM and RNN approaches to URL classification
- **Key Contributions**: Sequential character analysis, deep learning methods
- **Relevance**: Inspired feature extraction patterns and sequence analysis

### 2. Detecting Malicious URLs via a Keyword-Based Convolutional Gated-Recurrent-Unit Neural Network
- **Focus**: CNN-GRU hybrid architecture
- **Key Contributions**: Keyword extraction, attention mechanisms
- **Relevance**: Suspicious keyword lists, pattern recognition techniques

### 3. Malicious URL Detection Using Convolutional Neural Networks
- **Focus**: CNN-based URL analysis
- **Key Contributions**: Character-level features, convolution patterns
- **Relevance**: Feature engineering approaches

### 4. Entropy-based Phishing Detection Methods
- **Focus**: Statistical entropy analysis
- **Key Contributions**: Shannon entropy, randomness detection
- **Relevance**: Core statistical features (entropy, KL divergence, KS test)

### 5-8. Additional Research Papers
- Various approaches to URL feature extraction
- Machine learning methodologies
- Phishing pattern analysis
- Statistical feature engineering
